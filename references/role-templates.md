# Section Order & Metrics by Target Role — 职业方向模板

Different roles need different emphasis. Adjust section order and bullet
metrics accordingly.

---

## Tech — 技术类

### SWE / Engineer (default)
- **Order**: Header → Education → Skills → Experience → Publications
- **Key metrics**: latency reduction, QPS, uptime %, code coverage, users served

### Data Science / Analytics
- **Order**: Header → Education → Skills → Experience → Publications
- **Key metrics**: model accuracy, AUC, cost savings $, data volume processed, F1 score

### DevOps / SRE / Infrastructure
- **Order**: Header → Education → Skills → Experience → Certifications
- **Key metrics**: uptime %, deploy frequency, MTTR, infra cost savings $, incident count

### Cybersecurity
- **Order**: Header → Education → Certifications → Experience → Skills
- **Certs critical**: CISSP, CEH, OSCP, etc.
- **Key metrics**: incidents handled, vulnerabilities found, compliance %, response time

### Product Manager
- **Order**: Header → Education → Experience → Projects → Skills
- **Key metrics**: DAU/MAU, launch metrics, A/B test results, feature adoption rate, NPS

### Design / Creative
- **Order**: Header → Education → Experience → Projects → Skills
- **Portfolio link in contact line is critical**
- **Key metrics**: users impacted, design systems built, accessibility score, task completion rate

## Finance — 金融类

### Quant / Finance
- **Order**: Header → Education → Skills → Experience → Publications
- **Key metrics**: Sharpe ratio, PnL, max drawdown, alpha, latency, AUM

### Accounting / Audit
- **Order**: Header → Education → Certifications → Experience → Skills
- **CPA/CFA up front**
- **Key metrics**: audits completed, $ managed, compliance %, error reduction

## Business — 商业类

### Marketing / Growth
- **Order**: Header → Education → Experience → Skills → Awards
- **Key metrics**: ROI, conversion rate, CAC, LTV, engagement %, campaign reach, impressions

### Consulting / Strategy
- **Order**: Header → Education → Experience → Leadership → Skills
- **Key metrics**: revenue impact $, team size led, % improvement, client count, project count

### Sales / Business Development
- **Order**: Header → Education → Experience → Skills → Awards
- **Key metrics**: revenue generated $, deals closed, quota attainment %, client retention rate

### Operations / Supply Chain
- **Order**: Header → Education → Experience → Skills → Certifications
- **Key metrics**: cost reduction %, efficiency gain, throughput, error rate, cycle time

### Real Estate
- **Order**: Header → Education → Licenses → Experience → Skills
- **Key metrics**: deals closed $, properties managed, occupancy rate %, portfolio value

## Professional Services — 专业服务类

### Legal
- **Order**: Header → Education → Experience → Bar Admissions → Publications
- **Key metrics**: cases handled, deal value $, compliance rate, policies drafted, win rate

### Human Resources
- **Order**: Header → Education → Experience → Skills → Certifications
- **Key metrics**: headcount managed, time-to-hire, retention rate, programs launched, training hours

### Hospitality / Event Management
- **Order**: Header → Education → Experience → Skills → Certifications
- **Key metrics**: events managed, attendees, budget $, satisfaction score, revenue generated

## Academic / Research — 学术研究类

### Academic / Research / Teaching (university)
- **Order**: Header → Education → Publications → Research Experience → Teaching → Awards
- **Publications move UP — it's the main selling point**
- **Key metrics**: papers published, citations, h-index, grants awarded $, courses taught

### Education / Teaching (non-academic)
- **Order**: Header → Education → Teaching Experience → Skills → Awards
- **Key metrics**: students taught, pass rate %, curriculum designed, satisfaction score

### Biotech / Pharma Research
- **Order**: Header → Education → Publications → Experience → Skills
- **Key metrics**: papers, patents, pipeline stage, assay throughput, grant funding $

## Engineering — 工程类

### Architecture / Civil Engineering
- **Order**: Header → Education → Experience → Projects → Certifications
- **Key metrics**: project budget $, sq ft designed, LEED rating, team size, on-time delivery %

### Mechanical / Electrical / Hardware Engineering
- **Order**: Header → Education → Experience → Projects → Skills
- **Key metrics**: patents filed, cost reduction %, yield improvement, units shipped, BOM savings

## Media & Public — 媒体与公共类

### Journalism / Media
- **Order**: Header → Education → Experience → Portfolio → Skills
- **Portfolio link critical**
- **Key metrics**: articles published, readership, pageviews, awards, investigations completed

### Government / Public Policy
- **Order**: Header → Education → Experience → Publications → Skills
- **Key metrics**: policy impact, constituents served, budget managed $, legislation drafted

### Nonprofit / Social Impact
- **Order**: Header → Education → Experience → Volunteer → Skills
- **Key metrics**: funds raised $, people served, programs launched, grant success rate

## Special Cases — 特殊情况

### Junior / Intern (thin resume — any field)
- **Order**: Header → Education → Skills → Projects → Awards/Activities
- **Projects replace Experience; expand Education and Skills**
- See SKILL.md "Thin Resume Strategy" section

### Other / General
- **Order**: Header → Education → Experience → Skills → Awards
- **Ask user**: "What does your field value most?" and adapt accordingly
- **When in doubt**: lead with strongest section, end with weakest

---

## How to use this guide

```
1. Identify user's target role from the list above
2. Set section order accordingly
3. Write bullets emphasizing the listed key metrics
4. If role not listed → use "Other / General" and ask user what matters
5. If user applies to MULTIPLE roles → generate separate versions
```
